CNN Crawler and Search Engine

The CNN Crawler and search engine was created in 3 parts: 1st, the query suggestion was powered by a list of all main wikipedia page titles. These titles were loaded into a custom Trie data structure that allows returning search results in milliseconds. 2nd, the static NBA player results were generated by creating a PHP endpoint on an AWS EC2 instance which reads a list of NBA player stats on an RDS instance. 3rd, the search results were pulled from a database that was populated by a background web crawler. This web crawler (which can be accessed through the admin dashboard) scrapes through cnn.com and bleacherreport.com for titles + links. All the titles with their matching URL's are added to the database while URL's are added to a queue to be processed later.

Github Repo: CNN Scraper (C#)

MoeBot

MoeBot is a series of applications that perform a variety of tasks across the MoeBot website and Discord. The Discord interface responds to commands from Discord users, performing actions they request such as posting random images, posting jokes, and so on. MoeBot can also perform actions without a request such as preventing new users from posting in public servers until they have read the rules.

The website side of MoeBot allows browsing content that was uploaded by users or MoeBot. Users can also upload content that MoeBot can pull from when someone requests a random image.

Stego

Stego is a client side application that hides text within any image using steganography. This is accomplished by uploading the image (client side only, no server) and encoding each bit of the text into the LSB (Least Significant Bit) of the image's pixels. This process hides the text inside the image without any visible alteration to the image itself since each pixel only changes by at most 1 value (with a range of 0 - 255). The text is decoded from the image by reading each LSB of each pixel and building up a list of 1's and 0's. These are then converted to text and displayed to the user.

Github Repo: Stego (HTML, CSS, JS)
